Para *actualizar Deepin* es necesario acceder desde el Centro de Control. Otra forma es accediendo a la `Terminal de Deepin` actualizando la base de datos.
<!--more-->

~~~
sudo apt update
~~~

Para continuar, ejecute este comando:

~~~
sudo apt upgrade
~~~
